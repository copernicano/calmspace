Timer Audio File
================

Per il timer visuale, aggiungi un file audio chiamato "timer-beep.mp3" in questa cartella.

Il file dovrebbe essere:
- Breve (0.5-1 secondo)
- Volume moderato
- Tono chiaro e non aggressivo (es. campana soft, ding delicato)
- Formato: MP3

Suggerimenti per ottenere un beep timer:
1. Generatore online: https://www.zapsplat.com/sound-effect-category/beeps-and-blips/
2. FreeSound: https://freesound.org/search/?q=timer+beep
3. Generare con Audacity (Generate > Tone, 440Hz, 0.5s)

Se il file non è presente, l'alert audio semplicemente non suonerà (nessun errore).
